Rights To Access

Application Access and User Roles Our application is designed for use by all garage staff, but access levels vary based on user roles

  • Administration ID: Only authorized administrators have full access to the application's features and functionality.
  • Linked User Access: Linked user can access specific features, such as billing, through a unique link provided by the administration.
  • Linked User Login Process:
    1. Receive the link from the administrator.
    2. Click on the link and log in.
    3. Enter the generated code to access the linked user interface.
    4. Utilize the designated features and functions.

    • This structure ensures efficient management and security of your garage's operations.
